ld &  fret,
Function  f

One-dimensional minimization function.

Given as input the vectors P and n, and the function f, find the scalar a that minimizes f(P+an). Replace P by P+an. Replace n by an. Uses Brent's method.

Parameters
[in]pисходный вектор
[in]xiвектор, задающий направление минимизации
[in]fминимизируемая функция
[out]pвозвращает точку минимума f(p+a*xi). (а-по сути скаляр, находимый linmin)
[out]xiвозращает вектор минимизации a*xi
